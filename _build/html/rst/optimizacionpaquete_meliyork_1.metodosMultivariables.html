optimizacionpaquete_meliyork_1.metodosMultivariables.caminataAleatoria.caminata_aleatoria(f, x0, step, iter_max)[fuente]¶

Este método intenta encontrar un mínimo local de la función f realizando pasos aleatorios desde el punto inicial x0.

Parámetros:

  • f (function) – La función objetivo que se va a minimizar.

  • x0 (numpy.ndarray) – El punto inicial desde donde se empieza la caminata aleatoria.

  • step (float) – La magnitud máxima del paso aleatorio.

  • iter_max (int) – El número máximo de iteraciones a realizar.

Devuelve:

El punto donde se encontró el mínimo local.

Tipo del valor devuelto:

numpy.ndarray

Muestra:

ValueError – Si x0 no es un numpy.ndarray.

Ejemplo:

>>> import numpy as np
>>> def f(x):
>>>     return np.sum(x**2)
>>> x0 = np.array([1.0, 1.0])
>>> step = 0.1
>>> iter_max = 1000
>>> result = caminata_aleatoria(f, x0, step, iter_max)
>>> print(result)

optimizacionpaquete_meliyork_1.metodosMultivariables.cauchy module¶

optimizacionpaquete_meliyork_1.metodosMultivariables.cauchy.cauchy(f, x0, epsilon1, epsilon2, maxiter, metodo)[fuente]¶

Este método intenta encontrar un mínimo local de la función f usando el gradiente descendente y una búsqueda de línea con el método especificado.

Parámetros:

  • f (function) – La función objetivo que se va a minimizar.

  • x0 (numpy.ndarray) – El punto inicial desde donde comienza la optimización.

  • epsilon1 (float) – El umbral para la norma del gradiente bajo el cual se considera que la solución ha convergido.

  • epsilon2 (float) – El umbral para la norma del cambio relativo en xk bajo el cual se considera que la solución ha convergido.

  • maxiter (int) – El número máximo de iteraciones.

  • metodo (function) – El método de búsqueda de línea a utilizar.

Devuelve:

El punto donde se encontró el mínimo local.

Tipo del valor devuelto:

numpy.ndarray

Muestra:

ValueError – Si x0 no es un numpy.ndarray.

Ejemplo:

>>> import numpy as np
>>> def f(x):
>>>     return np.sum(x**2)
>>> def fibonacci_search(f, e, a, b):
>>>     L = b - a
>>>     fib = [0, 1]
>>>     while len(fib) <= e + 2:
>>>         fib.append(fib[-1] + fib[-2])
>>>     k = 2
>>>     while k < e:
>>>         Lk = (fib[e - k + 2] / fib[e + 2]) * L
>>>         x1 = a + Lk
>>>         x2 = b - Lk
>>>         fx1 = f(x1)
>>>         fx2 = f(x2)
>>>         if fx1 < fx2:
>>>             b = x2
>>>         elif fx1 > fx2:
>>>             a = x1
>>>         elif fx1 == fx2:
>>>             a = x1
>>>             b = x2
>>>         k += 1
>>>     return (a + b) / 2
>>> x0=np.array([0.0, 0.0])
>>> epsilon1=0.001
>>> epsilon2=0.001
>>> max_iter=100
>>> alpha=0.2
>>> result = print(cauchy(f, x0, epsilon1, epsilon2, max_iter, fibonacci_search))
>>> print(result)

  • x_initial (list or numpy.ndarray) – El punto inicial desde donde comienza la optimización.

  • delta (list or numpy.ndarray) – El tamaño del paso para la búsqueda exploratoria.

  • alpha (float) – El factor de reducción para el tamaño del paso.

  • epsilon (float) – El umbral para determinar la convergencia.

Devuelve:

El punto donde se encontró el mínimo local.

Tipo del valor devuelto:

numpy.ndarray

Muestra:

ValueError – Si x_initial o delta no son listas o numpy.ndarrays.

Ejemplo:

>>> import numpy as np
>>> def himmelblau(x):
>>>     return (x[0]**2 + x[1] - 11)**2 + (x[0] + x[1]**2 - 7)**2
>>> x_initial = [-5, -2.5]
>>> delta = [0.5, 0.25]

>>> epsilon = 0.1
>>> result = hooke_jeeves(himmelblau, x_initial, delta, alpha, epsilon)

optimizacionpaquete_meliyork_1.metodosMultivariables.newton module¶

optimizacionpaquete_meliyork_1.metodosMultivariables.newton.newton(f, x0, epsilon1, epsilon2, maxiter, metodo)[fuente]¶

Este método intenta encontrar un mínimo local de la función f utilizando el método de Newton, que emplea tanto el gradiente como la matriz Hessiana de la función objetivo.

Parámetros:

  • f (function) – La función objetivo.

  • x0 (list or numpy.ndarray) – Punto inicial.

  • epsilon1 (float) – Criterio de convergencia basado en el gradiente.

  • epsilon2 (float) – Criterio de convergencia basado en el cambio en las variables.

  • maxiter (int) – Número máximo de iteraciones permitidas.

  • metodo (function) – Método de búsqueda de línea para determinar el paso óptimo.

Devuelve:

El punto donde se encontró el mínimo local.

Tipo del valor devuelto:

numpy.ndarray

Ejemplo:

>>> import numpy as np
>>> def rosenbrock(x):
>>>     return sum(100.0*(x[1:]-x[:-1]**2.0)**2.0 + (1-x[:-1])**2.0)
>>> def fibonacci_search(f, e, a, b):
>>>     L = b - a
>>>     fib = [0, 1]
>>>     while len(fib) <= e +2:
>>>         fib.append(fib[-1] + fib[-2])
>>>     k = 2
>>>     while k < e:
>>>         Lk = (fib[e - k + 2] / fib[e+ 2]) * L
>>>         x1 = a + Lk
>>>         x2 = b - Lk
>>>         fx1 = f(x1)
>>>         fx2 = f(x2)
>>>         if fx1 < fx2:
>>>             b = x2
>>>         elif fx1 > fx2:
>>>             a = x1
>>>         elif fx1 == fx2:
>>>             a=x1
>>>             b=x2
>>>         k += 1
>>>     return (a+b)/2
>>> x0=np.array([0.0, 0.0])
>>> epsilon1=0.001
>>> epsilon2=0.001
>>> max_iter=100
>>> result = newton(himmelblau, x0, epsilon1, epsilon2, 1000, fibonacci_search)
>>> print(f"Resultado: x = {result}, f(x) = {rosenbrock(result)}")
