optimizacionpaquete_meliyork_1.metodosUnivariables.biseccion.biseccion(f, e, a, b)[fuente]¶

Realiza la búsqueda de la raíz de la derivada de la función f utilizando el método de bisección.

Este método encuentra un punto donde la primera derivada de la función f es cero, lo cual puede indicar un máximo o un mínimo local.

Parámetros:

  • f (function) – La función objetivo

  • e (float) – La tolerancia para el criterio de convergencia.

  • a (float) – El límite inferior del intervalo de búsqueda.

  • b (float) – El límite superior del intervalo de búsqueda.

Devuelve:

El punto donde la primera derivada de f es cero.

Tipo del valor devuelto:

float

Ejemplo:

>>> import numpy as np
>>> def funcion_objetivo(x):
>>>     return (x - 2)**2
>>> resultado = biseccion(funcion_objetivo, 1e-5, 0, 4)
>>> print(resultado)

optimizacionpaquete_meliyork_1.metodosUnivariables.busquedaDorada.busquedaDorada(funcion, e: float, a: float = None, b: float = None) → float[fuente]¶

Encuentra el mínimo de una función utilizando el método de búsqueda dorada.

La búsqueda dorada es un método de optimización unidimensional basado en la proporción áurea para reducir el intervalo de búsqueda de manera eficiente.

Parámetros:

  • funcion (function) – La función objetivo.

  • e (float) – La tolerancia para el criterio de convergencia del método.

  • a (float, opcional) – El límite inferior del intervalo de búsqueda. Si no se proporciona, se debe especificar.

  • b (float, opcional) – El límite superior del intervalo de búsqueda. Si no se proporciona, se debe especificar.

Devuelve:

El punto en el intervalo ([a, b]) donde la función tiene su mínimo.

Tipo del valor devuelto:

float

Ejemplo:

>>> import numpy as np
>>> def funcion_objetivo(x):
>>>     return (x - 2)**2
>>> minimo = busquedaDorada(funcion_objetivo, 1e-5, 0, 4)
>>> print(f"Resultado: x = {minimo}, f(x) = {funcion_objetivo(minimo)}")
>>>

optimizacionpaquete_meliyork_1.metodosUnivariables.intervalHalving.interval_halving_method(f, e, a, b)[fuente]¶

Encuentra el mínimo de una función unidimensional utilizando el método de reducción del intervalo.

El método de reducción del intervalo, o Interval Halving, busca minimizar una función unidimensional reduciendo el intervalo de búsqueda en cada iteración, eligiendo entre dos puntos que se encuentran a la mitad de la longitud del intervalo.

Parámetros:

  • f (function) – La función objetivo.

  • e (float) – La tolerancia para la longitud del intervalo de búsqueda, el proceso se detiene cuando (|b - a| < e).

  • a (float) – El límite inferior del intervalo de búsqueda.

  • b (float) – El límite superior del intervalo de búsqueda.

Devuelve:

El punto en el intervalo ([a, b]) donde la función tiene su mínimo.

Tipo del valor devuelto:

float

Ejemplo:

>>> def funcion_objetivo(x):
>>>     return (x - 2)**2
>>> minimo = interval_halving_method(funcion_objetivo, 1e-5, 0, 4)
>>> print(f"Resultado: x = {minimo}, f(x) = {funcion_objetivo(minimo)}")

optimizacionpaquete_meliyork_1.metodosUnivariables.newtonRaphson module¶

optimizacionpaquete_meliyork_1.metodosUnivariables.newtonRaphson.newton_raphson(x_0, f, E)[fuente]¶

Encuentra una raíz de una función unidimensional utilizando el método de Newton-Raphson.

El método de Newton-Raphson es un método iterativo para encontrar soluciones de ecuaciones no lineales. En cada iteración, el método utiliza la derivada de la función para aproximar una mejor solución a la raíz de la ecuación.

Parámetros:

  • x_0 (float) – El valor inicial para el punto de partida del método iterativo.

  • f (function) – La función objetivo.

  • E (float) – La tolerancia para el criterio de convergencia, el proceso se detiene cuando (|f'(x_{ ext{next}})| < E).

Devuelve:

El valor de (x) que aproxima una raíz de la función.

Tipo del valor devuelto:

float

Ejemplo:

>>> def funcion_objetivo(x):
>>>     return x**3 - x - 2
>>> raiz = newton_raphson(1.0, funcion_objetivo, 1e-5)
>>> print(f"Raíz encontrada: x = {raiz}, f(x) = {funcion_objetivo(raiz)}")

optimizacionpaquete_meliyork_1.metodosUnivariables.secante module¶

optimizacionpaquete_meliyork_1.metodosUnivariables.secante.secante(f, e, a, b)[fuente]¶

Encuentra una raíz de una función unidimensional utilizando el método de la secante.

El método de la secante es una técnica iterativa para encontrar soluciones de ecuaciones no lineales. A diferencia del método de Newton-Raphson, la secante no requiere el cálculo de la derivada, sino que utiliza una aproximación basada en dos puntos previos.

Parámetros:

  • f (function) – La función para la cual se busca una raíz.

  • e (float) – La tolerancia para el criterio de convergencia.

  • a (float) – El límite inferior del intervalo de búsqueda.

  • b (float) – El límite superior del intervalo de búsqueda.

Devuelve:

El valor de (x) que aproxima una raíz de la función.

Tipo del valor devuelto:

float

Ejemplo:

>>> def funcion_objetivo(x):
>>>     return x**3 - x - 2
>>> raiz = secante(funcion_objetivo, 1e-5, 1.0, 2.0)
>>> print(f"Raíz encontrada: x = {raiz}, f(x) = {funcion_objetivo(raiz)}")
